Descripción de dato

Se le solicitan los datos escenciales para regiistrar el tipo del proceso que tendrá cada marco de trabajo. Para ello debemos ingresar los siguientes datos.

Codigo: El codigo será el identificador unico que tendrá cada tipo de proceso.
Nombre del tipo: Este nombre corresponde al tipo del procesos como tal.
Descripción: La Descripción puede ser larga o corta, es simplemente lo que el tipo del proceso realiza
Estado: el estado activa o inactiva el tipo del proceso para ser o no asignado.


Operaciones del navegador

el navegador cuenta con diversas opcones las cuales se describen a continuación:
1. Escribir un nuevo registro: Permite crear nuevos registros si el usuario tiene permisos.
2. Cambiar:permite modificar un registro.
3. Guardar:permite guardar un registro modificado.
4. Cancelar:cancela las acciones que se realizaron en los cambios.
5. Eliminar:cambia el estado de los datos a 0.
6. Actualizar tabla: permite Actualizar la tabla de los datos.
