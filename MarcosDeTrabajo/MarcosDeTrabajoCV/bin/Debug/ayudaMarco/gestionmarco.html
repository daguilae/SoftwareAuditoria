Objetivo

El objetivo del mantenimiento de Gestión de Marcos de Trabajo es poder registar los procesos y objetivos de un marco de trabajo en específico. De esta manera cada marco de trabajo puede estar personalizado.


Pestaña Procesos

La pestaña de procesos esta estructurada de la siguiente manera (como se observa en la imágen). En esta captura podemos encontrar ejemplos de que datos ingresar.

La pestaña de objetivos esta estructurada de la siguiente manera (como se observa en la imágen). En esta captura podemos encontrar ejemplos de que datos ingresar.


Barra de Navegación

La barra de navegación nos permite realizar diferentes operaciones dentro del mantenimiento. Las opciones son las siguiente:

    1. NUEVO: insertar un nuevo registro
    2. MODIFICAR: modificar un registro existente
    3. AGREGAR: guardar el nuevo registro
    4. CANCELAR: cancelar una operación
    5. ELIMINAR: eliminar un registro existente
    6. AYUDA: botón para abrir esta ayuda
